GLYPH

Token-efficient serialization for AI agents

JSON
{"action": "search", "query": "weather", "limit": 10}
~18 tokens
GLYPH
{action=search query=weather limit=10}
~12 tokens

33% fewer tokens. Same data.

The GLYPH Difference

  • ✗ Commas → ✓ Spaces
  • ✗ Colons → ✓ Equals
  • ✗ "quoted" keys → ✓ bare keys
  • ✗ true/false → ✓ t/f
  • ✗ null → ✓ ∅
  • ✗ Repeat keys → ✓ @tab tables
